- 초기 로그인 id, pw

ID : admin
PW : 1234


위의 계정을 통해 로그인을 하는 순간 계정DB가 
프로젝트 폴더 내에 생성됩니다.

EDIT 버튼을 누르면 id와 pw를 바꿀 수 있고, OK버튼을 누르면
바뀐정보로 계정DB가 갱신됩니다.

Address와 Schedule 또한 정보를 입력하고 NEW 버튼을 눌러
추가하는 순간 각각의 DB가 폴더 내에 생성됩니다.

각각의 폰번호와 일정 리스트에서 하나를 선택하고 DELETE버튼을
누르면 해당 정보가 삭제됩니다.

LOGOUT버튼을 누르면 로그아웃이 됩니다.
